HTML (HyperText Markup Language) es un lenguaje de marcado de texto. Se utiliza para darle una estructura al sitio web que estás visitando.


Generalmente las paginas web tienen la siguiente estructura: 

- Header 
- Main content | Sidebar
- footer

Index será la primera página al abrir un proyecto. 
Todas las subsecciones tendrán su propio .html

En el head van todos los archivos importantes para que nuestro proyecto funcione correctamente. 
Algunos como: Estilos, fuentes, descripciones, librerías… Es la parte no visible de nuestra página.
